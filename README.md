# Sauce Demo E-Commerce Automation Framework

A robust, maintainable UI automation testing framework built using the **Page Object Model (POM)** design pattern. This project validates the core end-to-end user journeys of the Swag Labs (Sauce Demo) application.

## 🚀 Tech Stack & Architecture
* **Language:** Java 11
* **Automation Tool:** Selenium WebDriver
* **Build Tool:** Maven
* **Design Pattern:** Page Object Model (POM) with Fluent Interface design for enhanced readability.
